O que é o Code Club?

O Code Club (em português: Clube do Código ou Clube de Programação) é uma rede mundial de atividades extra-curriculares gratuitas, completamente gerenciada por voluntários, com o objetivo de ensinar programação de computadores às crianças.

Criamos projetos para que os nossos voluntários possam ensinar programação como atividade extra-curricular em escolas, ou outros locais, como por exemplo uma biblioteca. Nossos projetos ensinam as crianças a programar através de jogos, animações e páginas internet. Os voluntários atuam em locais próximos de suas casa, uma hora por semana.

Estaremos constantemente criando novas aulas. Em cada série os alunos aprendem usando imaginação e criatividade a través da realização de projetos. Os cursos 1 e 2 utilizam a ferramenta Scratch para ensinar as bases de programação de computadores. O curso 3 introduz o desenvolvimento web usando HTML e CSS. No curso 4 nós ensinaremos Python e assim por diante.

O Code Club foi criado por Clare Sutcliffe e Linda Sandvik em Abril de 2012. Em 2013, Everton Hermann teve a ideia de trazer o Code Club para o Brasil

Nosso sonho é que cada escola do Brasil tenha um clube de programação onde as crianças possam aprender informática. Temos quase 200 mil escolas no Brasil, temos muito trabalho pela frente, mas acreditamos que isso seja possível!

Por que deveríamos ensinar crianças a programar?

O que as escolas não ensinam:

Saber programar é uma habilidade importante em um mundo digital. Hoje em dia não basta saber usar computadores, criar textos, planilhas, usar internet. As crianças também devem saber como as coisas funcionam.

Aprender a programar não útil apenas se você quiser ser um programador no futuro. Programar ajuda em outras habilidades, como resolver problemas, desenvolve o raciocínio lógico e contribui em outras matérias como ciências e matemática. Além disso programar é muito divertido!

Nosso projetos foram desenvolvidos por programadores e testados em 20 escolas piloto no Reino Unido. Pedimos para as crianças darem uma nota aos projetos: zero (muito chato), 100 (a coisa mais legal que já fiz na vida).

Nossos projetos obtiveram a nota de 92!

Isso significa que eles adoraram!!

Nossa Filosofia

A filosofia do Code Club é diversão, criatividade e aprendizagem pela descoberta. É importante que as crianças curtam o tempo que elas passam no Code Club e que eles não se sintam como se estivessem em mais uma aula da escola. Eles devem entender que eles são responsáveis pelo que o computador faz, e pode (e devem) fazer com que o computador faça o que eles querem, não contrário.

Outros benefícios do Code Club como aprender a pensar de uma maneia computacional ou desenvolver a capacidade de programar, são objetivos secundários. Mesmo assim, as crianças irão assimilar todas estas habilidades maravilhosas no decorrer das projetos de uma maneira muito mais eficiente que utilizando aulas tradicionais.
