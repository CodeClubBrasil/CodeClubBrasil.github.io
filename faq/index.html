Perguntas Frequentes

O que é o Code Club?

O Code Club (em português: Clube do Código ou Clube de Programação) é uma rede mundial de atividades extra-curriculares gratuitas, completamente gerenciada por voluntários, com o objetivo de ensinar programação de computadores às crianças.

O que é ensinado no Code Club?

A programação é apenas um meio para que o aluno desenvolva sua lógica, criatividade e imaginação, além de entender como o mundo digital funciona. Existem 4 módulos: os módulos 1 e 2 utilizam a ferramenta Scratch para ensinar as bases da programação de computadores de uma forma lúdica. O módulo 3 introduz o desenvolvimento web usando HTML e CSS. No módulos 4, nós ensinaremos Python

Participando como voluntário

Como posso me tornar um voluntário?

É possível participar como voluntário de três maneiras:
- Montando um clube;
- Sendo monitor de um clube já existente; ou,
- Ajudando na tradução do material didático e vídeos do projeto.

Para todos os casos, comece conectando-se como voluntário em nosso site dos voluntários - é preciso conectar-se com uma conta do Gmail ou Facebook.

Como faço para criar um novo clube?

1) Conecte-se ao nosso site dos voluntários.
2) Cadastre seu clube utilizando o seguinte formulário.
3) Com o Clube criado, acesse o material didático no site de voluntários através do link Aulas presente no menu superior. Já temos o material para o primeiro módulo pronto (um trimestre), e aos poucos estamos disponibilizando os outros módulos.

Para mais informações sobre a criação de novos clubes, clique aqui.

Como faço para encontrar um clube?

Através do nosso mapa você pode encontrar um Code Club existente perto de você e fazer contato com o seu responsável, ou procurar outras pessoas da sua região interessadas no projeto através de nosso grupo no Facebook. Também disponibilizamos material gráfico para auxiliá-lo na apresentação do projeto em escolas que você queira começar um novo clube.

Como posso ajudar na tradução?

Temos um grupo no Facebook para discutir a tradução do material e um post em nosso Blog do Voluntário com um guia de uso do Github, que é a ferramenta que usamos para organizar as contribuições com a tradução.

É preciso ser expert em programação para ser voluntário?

Não é necessário ser um especialista em programação, tanto que as aulas são destinadas às crianças. Basta estar interessado e sentir-se confiante com o material, mas não é nada difícil. Qualquer dúvida você pode contar conosco!

Posso cobrar pelas aulas usando o material do Code Club?

Não. O projeto é baseado no voluntariado, a principal condição é que as atividades usando o material do Code Club sejam gratuitas.

Posso usar o material do Code Club em uma escola privada?

Sim. Desde que as atividades envolvendo o material do Code Club sejam gratuitas.

Posso criar um Clube em casa?

Um clube pode ser criado em qualquer lugar: em casa, numa biblioteca, em um centro cultural, etc.

Posso criar mais de um clube?

Claro! Um voluntário pode atuar em vários clubes, desde que tenha tempo disponível.

Estrutura necessária

É preciso internet durante as aulas?

Não. Existe uma versão offline do Scratch que pode ser baixada.

Onde posso encontrar os materiais?

Clique aqui para visualizar uma demostração do material. O material restante será enviado aos voluntários, assim que eles estiverem cadastrados.

Quais softwares preciso utilizar?

Os dois primeiros trimestres utilizam o Scratch. Ele está disponível em versão online e offline.

Quanto custa?

O material do Code Club é e sempre será gratuito! Este é um dos nossos princípios fundamentais e pretendemos mantê-lo desta forma. Code Clubs são gerenciados por voluntários, portanto não cobre nenhum tipo de remuneração nos clubes.
