Perguntas Frequentes

O que é o Code Club?

O Code Club (em português: Clube do Código ou Clube de Programação) é uma rede mundial de atividades extra-curriculares gratuitas, completamente gerenciada por voluntários, com o objetivo de ensinar programação de computadores às crianças.

Participando como voluntário

Como faço para me cadastrar?

Ainda não temos um sistema automático de cadastro de usuários.Entre em contato conosco por e-mail para que você seja inserido em nossa lista de voluntários.

Quero ser voluntário como faço para encontrar um clube?

Use a internet e redes sociais para descobrir se já existem clubes próximos de você. Você também pode entrar em contato conosco informando a cidade e tentaremos encontrar um clube próximo de você. Ou então entre em contato com uma escola ou associação, apresente o projeto e crie um novo clube.

É preciso ser expert em programação para ser voluntário?

Não necessário ser um especialista em programação, tanto que as aulas são destinadas às crianças. Basta estar interessado e sintir-se confiante com o material, mas não é nada difícil.

O Code Club fornece algum treinamento para voluntários?

Por enquanto não temos nem um tipo de treinamento. Mas não necessário ser um especialista em programação, tanto que as aulas são destinadas às crianças.

Posso cobrar pelas aulas usando o material do Code Club?

Não. O projeto baseado no voluntariado, a principal condição é que as atividades usando o material do Code Club sejam gratuitas.

Posso usar o material do Code Club em uma escola privada?

Sim. Desde que as atividades envolvendo o material do Code Club sejam gratuitas.

Posso criar um Clube em casa?

Um clube pode ser criado em qualquer lugar em casa, numa biblioteca em um centro cultural, etc

Estrutura necessária

É preciso internet durante as aulas?

Não. Existe uma versão offline do Scratch que pode ser baixada.

Onde posso encontrar os materiais?

Os materiais são enviados aos voluntários assim que eles estiverem cadastrados.

Quais softwares preciso utilizar?

Os dois primeiros trimestres utilizam o Scratch. Ele está disponível em versão online e offline.

Quanto custa?

O Scratch é gratuíto.
